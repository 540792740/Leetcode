#### ts 背景

- js 是动态变量
- ts 是静态变量，如果一个变量是静态变量，不仅类型不能修改，而且属性和方法也是确定的
- 所以编辑器在使用静态类型时候，就会有很好的语法提示

#### install

- sudo npm install -g typescript
- sudo npm install -g @types/node ts 的 node 运行环境
